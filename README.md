# Proyecto Pomodoro React Native
Aplicacion pomodoro creada con Expo en React native

---
## Componentes
* Header
* Timer
---
## Tecnologias
* expo: ~49.0.15
* expo-status-bar: ~1.6.0
* react: 18.2.0
* react-native: 0.72.6
* expo-av: ~13.4.1
---

# Instalacion
> git clone https://github.com/tomassale/Pomodoro
> cd Pomodoro
> npx install
> npx expo start
